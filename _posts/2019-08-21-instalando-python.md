Você conhece a linguagem queridinha dos cientistas de dados ?


Python é uma linguagem de programação flexível e versátil que pode ser utilizada em muitos em muitos casos, com ênfase em scripts, automação análise de dados, machine learning e até desenvolvimento back- end. Apareceu ao mundo em 1991 com seu nome sendo inspirado no de comédia Monty Python, a time de desenvolvimento queria fazer do Python uma linguagem divertida de usar(objetivo alcançado!!). Existem duas [versões](https://wiki.python.org/moin/Python2orPython3) de python, a 2, que é mais antiga e a 3 que é a versão mais atual.


Este tutorial irá guiá-lo na instalação do Python 3 no Windows e na verificação da instalação via prompt.

## Passo 1: Baixar o instalador Python
1. Abra seu navegador e entre em [página de download](https://www.python.org/downloads/)
2. navegue até a sessão **Python Releases for Windows** e clique em **Latest Python 3 Release - Python 3.x.x.** (No momento em que escrevo, o mais recente é o Python 3.7.4.)
3. Role a página até a sessão **files** e selecione o instalador executável do Windows x86-64 para o instalador executável de 64 bits ou do Windows x86 para 32 bits.

## Passo 2: Executar o instalador
Depois de ter escolhido e baixado o instalador, basta clicar duas vezes para instalá-lo e uma página com essa deve aparecer.
![image-center]({{ site.url }}{{ site.baseurl }}/images/python1.jpeg){: .align-center}

**Importante!!** Certifique de marcar a caixa de diálogo **Add Python 3.7** to path  conforme mostrado, para garantir que o interpretador seja colocado em seu caminho de variáveis de ambiente.

Em seguida, basta clicar em **Install Now**, e pronto, em poucos minutos depois, você deve ter uma instalação funcional do Python 3 em seu sistema. maravilha emotions mãos pra cima.

## Passo 2: Verificar sua instalação do Python
Finalmente, você pode testar sua nova versão do Python, abra um prompt de comando do windows e digite: *python --version* e deve aparecer algo parecido com isto.
![image-center]({{ site.url }}{{ site.baseurl }}/images/python2.jpeg){: .align-center}
