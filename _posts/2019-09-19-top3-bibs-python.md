Você conhece as bibliotecas que são uma mão na roda no mundo da ciência de dados?  Hoje vou ensinar você instalá-las e deixar sua máquina prontinha para os futuros trabalhos em [DS](http://www.cienciaedados.com/o-que-e-data-science/).


[**Aqui**]({{ "/posts/install-python/" | relative_url }}) você já aprendeu a instalar o Python,  e [**aqui**]({{ "/posts/hello/" | relative_url }}) você aprendeu fazer o primeiro programada em Python, agora instalaremos as top 5 bibliotecas Python para Ciências de Dados

A primeira biblioteca é a [**Numpy**](https://numpy.org/). Tradicionalmente se inicia os estudos com problemas de cunho de computação científica, nesse aspecto numpy é a melhor, possui objetos para manuseio de grandes matrizes multidimensionais, e um conjunto de funções matemáticas de alto nível.

O caminho mais rápido e mais fácil de instalar a biblioteca NumPy em sua máquina é usar o seguinte comando no prompt:

```sh
pip install numpy
```

Isso instalará a versão mais recente e mais estável do NumPy em sua máquina. Instalar através do pip é a maneira mais simples de instalar qualquer pacote Python.

A próxima biblioteca também com aspecto de computação cientifica é o [**SciPy**](https://www.scipy.org/). É baseada em NumPy, portanto, amplia seus recursos. A estrutura principal de dados do SciPy é novamente uma matriz multidimensional, implementada pela Numpy. O pacote contém ferramentas que ajudam a resolver problemas de álgebra linear, teoria das probabilidades, cálculo integral e muitas outras tarefas.

Para instalá-la digite no prompt, 

```sh
pip install scipy
```
